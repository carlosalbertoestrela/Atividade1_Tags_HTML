Algumas tags HTML e suas funções


  1. Tag '<p>':

    A tag '<p>' cria um parágrafo

  2. Tag '<a>':

    A tag '<a>' cria uma ancora para um link

  3. Tag '<meta>':

    recebe informações de metadados

  4. Tag '<header>':

    A tag '<header>'Cria um cabeçario para um bloco de conteúdo

  5. Tag '<div>':

    A tag '<div>' cria um bloco para agrupar elementos

  6. Tag '<H1> - <H6>:

    As tags '<H1> - <H6>' representam os níveis de títulos

  7. Tag '<dl>':

    A tag '<dl>' cria uma lista de descrção

  8. Tag '<ol>':

    A tag '<ol>' gera uma lista ordenada

  9. Tag '<ul>':

    A tag '<ul>' gera uma lista não ordenada

  10. Tag '<li>':

    A tag '<li>' gera um item numa lista

  11. Tag '<br>':

    A tag '<br>' gera uma quebra de linha

  12. Tag '<title>':

    A tag '<titel>' recebe o titulo da pagina

  13. Tag '<img>':

    A tag '<img>' permite likar uma imagem ao documento

  14. Tag '<b>':

    A tag '<b>' formata o texto em negrito

  15. Tag '<i>':

    A tag '<i>' rformata o texto em itálico
